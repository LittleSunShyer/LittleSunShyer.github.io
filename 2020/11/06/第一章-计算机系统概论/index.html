第一章 计算机系统概论

发表于2020-11-06|更新于2020-11-10
|阅读量:

计算机系统简介

软硬件概念

  • 硬件

    计算机的实体部分。

    由各种电子元件，各类光，电，机设备，比如主机，外设等。

  • 软件

    具有特殊功能的程序。

    通常存放在计算机的主存或辅存。

    软件的地位和作用已经成为评价计算机系统性能好坏的重要标志。

软件性能的发挥必须依托硬件，计算机的性能好坏取决于软件和硬件功能的总和。

计算机组成和计算机体系结构

  • 计算机体系结构

    计算机概念性的结构与功能特性。

    比如：指令集，数据类型，存储器寻址技术，I/O机理等

  • 计算机组成

    对计算机体系结构的实现，包含了硬件细节。

计算机体系结构是概念性的，组成是对概念性的功能的实现细节。

计算机的基本组成

冯诺伊曼计算机

  • 计算机由**运算器，存储器，控制器，输入设备和输出设备**五大部件组成。
  • 指令**和数据**以同等地位存放于存储器内，并可按地址寻访。
  • 指令**和数据**均用二进制数表示。
  • 指令由*操作码和地址码**组成，操作码用来表示操作的性质*，地址码用来表示操作数在存储器中的位置。
  • 指令在存储器内按顺序存放。通常，指令是顺序执行的，在特定条件下，可根据运算结果或根据设定的条件改变执行顺序。
  • 机器**以运算器为中心**，输入输出设备与存储器间的数据传送通过运算器完成。

现代计算机

  • 现代计算机**以存储器为中心**。

五大部件功能

  • 运算器

    完成算术运算和逻辑运算，将运算的中间结果暂存在运算器内。

  • 存储器

    存放数据和程序。

  • 控制器

    用来控制，指挥程序和数据的输入，运行以及处理运算结果。

  • 输入设备

    将信息转换为机器能识别的信息形式，比如键盘，鼠标。

  • 输出设备

    将机器运算结果转换为人们熟悉的信息形式，比如打印机输出，显示器输出等。

五大部件都由控制器指挥工作。

运算器和控制器统称为**中央处理器**（Central Processing Unit，CPU）。

输入/输出设备统称为**I/O设备**。

存储器有主存储器和辅助存储器，主存储器用来存放程序和数据，可以直接与CPU交换信息。

所以现代计算机主要由三大部分组成：CPU，I/O设备及主存储器（Main Memory，MM），CPU与主存储器合称主机，I/O设备可称为外部设备。

计算机工作步骤

计算机工作过程

  • 主存储器

    包括存储器M，各种逻辑部件及控制电路等。

    存储体由许多**存储单元**组成，每个存储单元又包含若干个存储元件（也叫存储基元，存储元），每个存储元件能寄存一位二进制代码0或1。

    1存储体 : N存储单元

    1存储单元 : N存储元件

    一个存储单元存储的一串二进制代码叫**存储字**，这串二进制代码的位数称为**存储字长**。

    存储字长可以是8，16，32位等。

    工作方式

    按存储单元的地址号来实现对存储字各位的存（写入），取（读出）。这种存取方式称为按地址存取方式，即按地址访问存储器（简称访存）。

    MAR（Memory Address Register）存储器地址寄存器。用来存放要访问的存储单元的地址，其位数对应存储单元的个数，比如MAR是10位，则有2^10个存储单元，记为1K。

    MDR（Memory Data Register）存储器数据寄存器。存放从存储体某单元取出的代码或准备往某存储单元存入的代码，其位数与存储字长相等。

    早起的存储字长一般和机器的指令字长与数据字长相等。现在可以不相同，但都必须是字节的整数倍。

  • 运算器

    最少包括3个寄存器（现代计算机内部往往有通用寄存器组）和一个算术逻辑单元（ALU）。三个寄存器分别为ACC是累加器，MQ是乘商寄存器，X是操作数寄存器。

    不同机器的运算器结构是不同的。

    加法 减法 乘法 除法
    ACC 被加数/和 被减数/差 乘积高位 被除数/余数
    MQ 乘数/乘积低位 商
    X 加数 减数 被乘数 除数

  • 控制器

    控制器是计算机的神经中枢。

    1. 控制器首先命令存储器读出一条指令，称为取指过程（取指阶段）。
    2. 对这条指令进行分析，指出指令要完成的操作，按寻址特征指明操作数的地址，称为分析过程（分析阶段）。
    3. 根据操作数所在的地址以及指令的操作码完成某种操作，称为执行过程（执行阶段）。

控制器由程序计数器（Program Counter, PC），指令寄存器（Instruction Register, IR）以及控制单元（CU）组成：

  • PC用来存放当前欲执行指令的地址，它与主存的MAR之间有一条直接通路，且具有自动加1功能，即可以自动形成下一条指令的地址。
  • IR用来存放当前指令，IR内容来自主存的MDR。
  • CU用来分析当前指令需要完成的操作，发出各种微操作命令序列，用以控制所有被控对象。

  • I/O

    I/O子系统包括各种I/O设备及其相应的接口。每种I/O设备都由I/O接口与主机联系，它接收CU发出的各种控制命令，并完成相应操作。

计算机硬件的主要技术指标

  • 机器字长

    指CPU一次能处理数据的位数，通常与CPU的寄存器位数有关。

    字长越长，数的表示范围越大，精度越高。

    字长会影响机器的运算速度：如果字长较短，又要运算位数较多的数据，那得进行多次运算，运算速度自然也就慢。

    机器字长也会直接影响ALU，数据总线以及存储字长的位数。

  • 存储容量

    包括主存容量和辅存容量

    主存容量指主存中存放二进制代码的总位数：存储容量 = 存储单元个数 \ 存储字长*。

    MAR的位数反映存储单元的个数，MDR的位数反映了存储字长。

    比如MAR为16位，那么存储单元就有2^16个，一个存储单元对应一个存储字，所以有2^6K个存储字（1K=2^10）。MDR为32位，那么存储容量为2^16 * 32 = 2^21 = 2M位，一个字节是8位，所以存储容量为2^18个字节，也就是2^18B，256KB。

  • 运算速度

    早起用普通法：完成一次加法或乘法所需的时间来衡量运算速度。来衡量运算速度。

    之后用吉普森法。

    现在用MIPS（Million Instruction Per Second, 百万条指令每秒）作为计量单位。

    也可以用CPI（Cycle Per Instruction, 执行一条指令需要的时钟周期（机器主频的倒数）），FLOPS（Floating Point Operation Per Second, 浮点运算次数每秒）来衡量运算速度。
